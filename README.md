### ¡Qué bueno verte por acá!

Soy informático con experiencia en desarrollo backend, operación de sistemas y soporte
técnico.

Desarrollo APIs REST y aplicaciones web con Node.js, TypeScript, Express.js y
PostgreSQL, aplicando arquitecturas desacopladas y buenas prácticas de desarrollo.

Mi experiencia previa en operación de sistemas me permitió desarrollar una sólida
capacidad de análisis y resolución de problemas.

Actualmente continúo formándome en Java y Spring Boot para ampliar mis
conocimientos en el desarrollo backend.

## Tecnologías

Backend: Node.js, TypeScript, Express.js, APIs REST, Prisma ORM, TypeORM, autenticación JWT,  arquitectura en capas, validación de datos.

Bases de datos: PostgreSQL, MySQL, MongoDB.

Operación de Sistemas: Control-M, Linux, monitorización de procesos batch, gestión de incidencias, análisis de logs.

Herramientas: Docker, Git, GitHub, Vercel, Render.

Testing: Jest, Supertest, pruebas unitarias y de integración.

Frontend: React, Next.js, JavaScript, HTML5, CSS3, Tailwind CSS.
